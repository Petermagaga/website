Maternal Health Platform

A secure, API-driven maternal health system designed for mobile users and real-world data reliability.

Problem

Expectant mothers lacked a centralized, secure platform to manage health information via mobile devices.

Solution

Built a Django REST API with JWT authentication, integrated with Flutter and Firebase.

Architecture

  • Django REST Framework
  • JWT Authentication
  • PostgreSQL
  • Flutter Mobile App

Outcome

Delivered a scalable foundation capable of supporting secure maternal health workflows and future growth.
